## 💡 Overview

<a href="#">ChatGPT</a> has become the default reference point for AI assistance because it handles a wider range of tasks with genuine capability than any single-purpose tool — writing, code, research, analysis, math, translation, creative work, and explanation all function within the same interface without requiring different tools for different functions.

The <a href="#">advanced reasoning models</a> o1 and o3 apply extended, systematic thinking to complex problems in mathematics, science, logic, and coding. These models invest more time in difficult problems before responding and produce more accurate results for tasks that require methodical reasoning rather than pattern-based generation. <a href="#">Integrated web search</a> retrieves current information from the internet for questions requiring facts beyond the training knowledge cutoff, attributing sources clearly so retrieved information is distinguishable from model knowledge.

<a href="#">Canvas collaborative editing</a> opens a side-by-side document workspace where the AI and user work on the same text or code artifact iteratively — applying targeted changes to specific sections, tracking modifications with visible differences, and continuing the conversation alongside the shared document without the copy-paste cycle of standard chat. <a href="#">Custom GPTs</a> are specialized configurations with specific instructions, knowledge, and capabilities built for particular recurring workflows — accessible from the desktop application for consistent behavior without context re-establishment each session.

The <a href="#">code interpreter with Python execution</a> runs computation within conversations for data analysis, chart generation, file processing, and mathematical tasks that require actual execution rather than descriptive responses about what computation would show. <a href="#">DALL-E image generation</a> creates original images from text descriptions within the conversation interface, enabling creative and production image work without switching to a separate image generation tool.

## 💬 FAQ

<details>
<summary>What is Canvas and when should I use it?</summary>

Canvas opens a side-by-side editing workspace for documents or code where AI and user work on the same artifact. Use it for drafting and iteratively refining documents or code rather than for Q&A or research, where standard chat is more appropriate.

</details>

<details>
<summary>How does web search work in ChatGPT?</summary>

For queries requiring current information, ChatGPT searches the web and incorporates findings with source citations into the response. The feature activates automatically for relevant queries on plans that include web search access.

</details>

<details>
<summary>What are Custom GPTs and how do they differ from standard ChatGPT?</summary>

Custom GPTs are configurations with specific system instructions, custom knowledge, and defined capabilities created for particular recurring use cases. They maintain consistent behavior without requiring context re-establishment and can be shared with others.

</details>

<details>
<summary>Does ChatGPT save my conversation history?</summary>

Yes. All conversations save to the sidebar history and remain accessible. Individual conversations or all history can be deleted from settings. History can also be disabled to prevent saving entirely.

</details>
